# Калькулятор
Простое приложение-калькулятор с графическим интерфейсом на Python.

## Функциональность:
- Сложение, вычитание, умножение, деление.
- Обработка ошибок (деление на ноль, некорректный ввод).
- Очистка ввода.

## Запуск:
1. Убедитесь, что установлен Python 3.x.
2. Запустите файл `калькулятор.py`.
3. Используйте кнопки для ввода выражений.

## Пример использования:
- Введите: 5 + 3
- Нажмите "=" → результат: 8
